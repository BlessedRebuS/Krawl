Horizon Analytics API

Welcome to the Horizon Analytics platform. This endpoint provides access to system statistics and performance metrics.

GET /api/stats

Retrieve current system statistics including user activity, server load, and application performance.

Parameters:

  • period (optional): Time period for stats (hour, day, week, month)
  • format (optional): Response format (json, xml, csv)
  • include_details (optional): Include detailed breakdowns (true/false)
{ "status": "success", "timestamp": "2024-01-15T10:30:45Z", "data": { "users": { "total": 12458, "active_24h": 3421, "new_today": 89 }, "system": { "cpu_usage": 23.4, "memory_usage": 67.8, "disk_usage": 45.2, "network_in": "1.2 GB", "network_out": "890 MB" }, "api": { "requests_per_minute": 1240, "avg_response_time": 145, "error_rate": 0.02, "endpoints": { "/login": 450, "/dashboard": 320, "/api/data": 280, "/api/stats": 190 } } } }

POST /api/stats/reset

Reset statistics counters (requires admin privileges).

Example Request:

{ "api_key": "your_admin_key_here", "confirm": true }
