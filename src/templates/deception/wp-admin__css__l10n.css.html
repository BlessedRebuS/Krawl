WordPress Admin Localization Styles

This stylesheet provides localization support for WordPress admin interface elements.

Note: This file is part of the WordPress admin interface and should not be modified directly. Changes may be overwritten during updates.

Usage

This CSS file is automatically loaded by WordPress when admin pages are rendered. It contains styles for:

  • Language selector dropdowns
  • Translation strings in admin notices
  • RTL (Right-to-Left) language support
  • Date and time formatting in admin screens

Example Classes

.rtl - Applied to <html> or <body> for right-to-left languages

.wp-core-ui - Base class for WordPress admin UI components

.hide-if-no-js - Hides element when JavaScript is disabled

.screen-reader-text - Makes text accessible to screen readers but visually hidden

Variables (CSS Custom Properties)

:root {
    --wp-admin-primary: #21759b;
    --wp-admin-primary-dark: #1e6f93;
    --wp-admin-light: #f1f1f1;
    --wp-admin-gray-light: #ddd;
    --wp-admin-gray: #777;
    --wp-admin-danger: #d54e21;
    --wp-admin-success: #46b450;
    --wp-admin-warning: #f0a00e;
}

/* Language-specific overrides */
[dir="rtl"] {
    --wp-admin-margin-left: 0;
    --wp-admin-margin-right: 20px;
}

[dir="ltr"] {
    --wp-admin-margin-left: 20px;
    --wp-admin-margin-right: 0;
}
