Telescope API Documentation

Endpoint: /telescope/requests

Overview

This endpoint provides access to historical request logs captured by the Telescope monitoring system. It allows filtering, pagination, and detailed inspection of HTTP requests processed by the application.

⚠️ Security Notice: This endpoint may expose sensitive request data including headers, payloads, and authentication tokens. Access should be restricted to authorized personnel only in production environments.

Available Parameters

filter (string): Filter requests by URL pattern, method, or status code (e.g., "GET /api/* 200")

limit (integer): Maximum number of results to return (default: 50, max: 1000)

offset (integer): Number of results to skip for pagination

start_time (string): ISO 8601 timestamp for filtering requests after this time

end_time (string): ISO 8601 timestamp for filtering requests before this time

include_payload (boolean): Whether to include request/response bodies in the output

Example Usage

GET /telescope/requests?filter=POST+/api/users+201&limit=10&include_payload=true

Notes

  • Response times are measured in milliseconds
  • Sensitive data in payloads may be redacted based on security policies
  • Maximum response size is limited to 10MB
  • This endpoint requires the telescope:read permission
