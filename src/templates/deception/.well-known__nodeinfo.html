NodeInfo Instance Metadata

This endpoint provides standardized metadata about this Mastocompatible instance according to the NodeInfo 2.0 specification.

Protocol Versions

Supported NodeInfo versions:

  • http://nodeinfo.diaspora.software/ns/schema/2.0
  • http://nodeinfo.diaspora.software/ns/schema/1.0

Instance Metadata (NodeInfo 2.0)

Access the full metadata at: /nodeinfo/2.0

Note: This is a discovery endpoint only. Actual instance statistics are available via the versioned endpoints.

Related Endpoints

  • /nodeinfo - Legacy NodeInfo 1.0 discovery
  • /nodeinfo/2.0 - Full NodeInfo 2.0 metadata
  • /api/v1/instance - Mastodon instance info
  • /.well-known/webfinger - WebFinger endpoint
