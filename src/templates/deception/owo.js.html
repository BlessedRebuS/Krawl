owo.js - Cute Utility Functions

A lightweight JavaScript library for adding uwu-ification and other kawaii text transformations to web applications.

Installation

npm install owo.js

Usage

// Basic usage
const owo = require('owo.js');
console.log(owo('Hello world!')); // Outputs: Hewwo wowwd!

// Advanced options
owo.text('I love programming!', { 
    uwu: true,
    smiley: true,
    stutter: 0.3
}); // Outputs: I-I wuv pwogwamming!! ( >‿<)

API Reference

owo(text, options)

Transforms text with various kawaii effects.

  • text (string): The input text to transform
  • options (object): Configuration options
    • uwu (boolean): Apply uwu-ification (default: true)
    • smiley (boolean): Add random smileys (default: false)
    • stutter (number): Probability of stutter (0-1, default: 0)
    • wizard (boolean): Apply wizard-speak transformation (default: false)

Examples

owo('Hello there!'); 
// Returns: Hewwo thewe!

owo('Good morning!', { 
    uwu: true, 
    smiley: true 
}); 
// Returns: Gud mowning!! ( ' w ' )

owo('Test', { 
    stutter: 0.5, 
    wizard: true 
}); 
// Returns: T-t-test, discerning eye of newt!
