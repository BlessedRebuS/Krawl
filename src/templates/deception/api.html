Internal API Service

Welcome to the developer portal for our internal RESTful API. Use the endpoints below to interact with the system.

Available Endpoints

  • GET /api/users – Retrieve a list of user accounts.
  • POST /api/users – Create a new user account.
  • GET /api/users/:id – Get details for a specific user.
  • PUT /api/users/:id – Update user information.
  • DELETE /api/users/:id – Delete a user.
  • GET /api/logs – Fetch recent system logs (admin only).
  • POST /api/auth/login – Authenticate and obtain a session token.
