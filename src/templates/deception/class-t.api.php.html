Class T API Documentation

Welcome to the Class T Educational Resource Management API. This interface provides programmatic access to course data, student records, and administrative functions for educational institutions.

GET /class-t.api.php?action=get_courses
Retrieve a list of all available courses with their metadata.
Parameters: format=json|xml (default: json), limit=integer (default: 50)
GET /class-t.api.php?action=get_students&course_id=123
Get enrolled students for a specific course.
Required: course_id=integer. Optional: status=active|inactive
POST /class-t.api.php?action=enroll_student
Enroll a student in a course.
Required POST fields: student_id, course_id, enrollment_date (YYYY-MM-DD)
GET /class-t.api.php?action=get_grades&student_id=456
Required: student_id=integer. Optional: term=fall|spring|summer, year=YYYY
Important: This API requires authentication via API key in the X-API-Key header. Rate limiting: 100 requests per minute per IP. All data is encrypted in transit and at rest.
GET /class-t.api.php?debug=1
Debug endpoint for development purposes (disabled in production).
Returns system information and cached query performance metrics.
