WP Mail SMTP – Settings Endpoint

This is a simulated endpoint for demonstration purposes only.

The /wp-json/wpmailsmtp/v1/settings endpoint returns the current SMTP configuration in JSON format.

Example Response

{
    "status": "success",
    "data": {
        "mailer": "smtp",
        "smtp_host": "smtp.example.com",
        "smtp_port": 587,
        "smtp_encryption": "tls",
        "smtp_username": "user@example.com",
        "smtp_password": "********",
        "from_name": "Example Site",
        "from_email": "no-reply@example.com",
        "return_path": true
    }
}

To test the settings, you can use the WP Mail SMTP settings page in the WordPress admin dashboard.
