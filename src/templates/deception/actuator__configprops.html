Spring Boot Actuator - Configuration Properties

This endpoint exposes configuration properties from the Spring Environment.

Configuration Properties

server.port
8080
Server HTTP port.
spring.application.name
demo-application
Name of the application.
spring.datasource.url
jdbc:mysql://localhost:3306/demo_db
JDBC URL of the database.
spring.datasource.username
dbuser
Login username of the database.
spring.datasource.password
${SPRING_DATASOURCE_PASSWORD:****}
Login password of the database (masked).
spring.jpa.hibernate.ddl-auto
update
DDL mode. This is actually a shortcut for the "hibernate.hbm2ddl.auto" property.
management.endpoints.web.exposure.include
*
Comma-separated list of endpoint IDs that should be exposed.
management.endpoint.health.show-details
when-authorized
When to show full health details.
logging.level.org.springframework
INFO
Logging level for Spring framework.
logging.level.org.hibernate.SQL
DEBUG
Log all SQL queries from Hibernate.
