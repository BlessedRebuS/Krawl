Elasticsearch Cluster Settings API

This endpoint allows you to view and update cluster-wide settings.

GET /_cluster/settings

Current cluster settings:

{
  "persistent" : {
    "cluster" : {
      "routing" : {
        "allocation" : {
          "enable" : "all",
          "disk" : {
            "watermark" : {
              "low" : "85%",
              "high" : "90%",
              "flood_stage" : "95%"
            }
          }
        }
      }
    },
    "indices" : {
      "recovery" : {
        "max_bytes_per_sec" : "20mb"
      }
    }
  },
  "transient" : { }
}

To update settings, send a PUT request with JSON body:

PUT /_cluster/settings

Example request body:

{
  "persistent" : {
    "indices.recovery.max_bytes_per_sec" : "50mb"
  }
}
