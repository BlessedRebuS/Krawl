WebMention 1.0 Endpoint

This is the WebMention endpoint for receiving webmentions according to the W3C WebMention specification.

POST /wp-json/webmention/1.0

Accepts application/x-www-form-urlencoded data with source and target parameters.

Usage Example

curl -X POST \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "source=https://example.com/blog/post&target=https://yoursite.com/blog/post" \
  https://yoursite.com/wp-json/webmention/1.0

Response Codes

  • 202 Accepted: Webmention queued for processing
  • 400 Bad Request: Missing or invalid source/target parameters
  • 403 Forbidden: Target URL not allowed for webmentions
  • 404 Not Found: Source URL not found or not accessible
  • 405 Method Not Allowed: Only POST requests are accepted
  • 410 Gone: Endpoint no longer available
  • 429 Too Many Requests: Rate limit exceeded
  • 500 Internal Server Error: Server error during processing

Security Note

This endpoint implements rate limiting and validates all incoming URLs to prevent abuse. Only HTTP(S) URLs are accepted as sources and targets.
