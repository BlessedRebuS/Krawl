How to Use the Dashboard

  1. Log in using your corporate credentials at /admin/login.php.
  2. Navigate to the desired module via the sidebar.
  3. For API interactions, refer to /api/v1/ endpoints.

Common Issues

  • Login fails: Ensure your password is not expired. Contact IT Support.
  • API returns 500: Check the debug flag in /config/settings.xml.
  • Dashboard appears blank: Clear your browser cache or try a different browser.
