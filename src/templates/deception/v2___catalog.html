Docker Registry v2 API

Welcome to the Docker Registry HTTP API V2. This endpoint provides access to the registry's catalog functionality.

GET /v2/_catalog

List all repositories in the registry.

Parameters:
  • n (optional): Maximum number of repository names to return
  • last (optional): Return repositories after this value (for pagination)
Response: 
{
  "repositories": [
    "library/nginx",
    "library/mysql", 
    "library/redis",
    "library/postgres",
    "library/mongo",
    "app/frontend",
    "app/backend",
    "api/auth-service",
    "api/payment-service",
    "infra/database",
    "infra/cache",
    "monitoring/prometheus",
    "monitoring/grafana",
    "ci/cd-pipeline",
    "security/vuln-scanner"
  ]
}

Authentication

This registry requires authentication for push/pull operations. Catalog listing is publicly accessible.

Note: For security reasons, anonymous access to catalog may be disabled in production environments.
