Internal API v1 Documentation

Welcome to the internal API version 1. Use the following endpoints to interact with the service.

GET /v1/users
Retrieve a list of users. Supports search query parameter.
POST /v1/login
Authenticate a user. Expects JSON body with username and password.
GET /v1/data?id=
Fetch a record by ID. Vulnerable to SQL injection if id is not sanitized.
