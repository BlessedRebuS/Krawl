LiveWire WebSocket Service

● Service Status: Online

● Protocol: WebSocket (WS/WSS)

● Server: Laravel LiveWire v2.12.0

● PHP Version: 8.1.2

Available Endpoints

Primary WS Endpoint:
ws://your-domain.com/livewire/ws
Used for real-time component updates and DOM diffing
Secure WSS Endpoint:
wss://your-domain.com/livewire/ws
Encrypted WebSocket connection (requires valid SSL certificate)

Connection Parameters

Headers Required:
Upgrade: websocket
Connection: Upgrade
Sec-WebSocket-Key: [base64-encoded-random]
Sec-WebSocket-Version: 13
Cookie: laravel_session=[session-id]; XSRF-TOKEN=[token]
Message Format (JSON):
{
  "fingerprint": {
    "id": "component-hash",
    "name": "ComponentName",
    "method": "methodName",
    "params": ["param1", "param2"]
  },
  "updates": [
    {
      "type": "dom",
      "selector": "#element-id",
      "html": "
Updated content
"
    }
  ],
  "errors": [],
  "url": "/current-route"
}

Security Notes

⚠️ This endpoint requires authentication. Unauthorized connection attempts are logged and may trigger IP blocking.

● All WebSocket connections are rate-limited to 100 connections per minute per IP

● Message size is limited to 64KB to prevent DoS attacks

● Invalid JSON payloads are immediately rejected

Sample Connection (JavaScript)

// Initialize LiveWire WebSocket connection
const socket = new WebSocket('ws://your-domain.com/livewire/ws');

socket.onopen = function() {
    console.log('LiveWire WS connected');
    // Send authentication token
    socket.send(JSON.stringify({
        event: 'auth',
        token: document.querySelector('meta[name="csrf-token"]').content
    }));
};

socket.onmessage = function(event) {
    const data = JSON.parse(event.data);
    // Handle LiveWire updates
    LiveWire.receive(data);
};

socket.onclose = function() {
    console.log('LiveWire WS disconnected');
    // Attempt reconnection after 3 seconds
    setTimeout(() => {
        const socket = new WebSocket('ws://your-domain.com/livewire/ws');
    }, 3000);
};
