Internal REST API Documentation

Welcome to the internal API portal. Use the following endpoints to interact with the service.

GET /api/users
Retrieve a list of users. Optional query parameters: limit, offset, search.
[
  {"id":1,"username":"alice","email":"alice@example.com","role":"admin"},
  {"id":2,"username":"bob","email":"bob@example.com","role":"user"},
  {"id":3,"username":"charlie","email":"charlie@example.com","role":"user"}
]
POST /api/users
Create a new user. Required fields: username, email, password.






GET /api/products?id=
Fetch product details by ID. Example: /api/products?id=42 returns:
{
  "id":42,
  "name":"Widget Pro",
  "price":29.99,
  "stock":150,
  "description":"High-performance widget for professional use."
}
